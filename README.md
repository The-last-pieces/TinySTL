# TinySTL

对STL的部分复现和扩展.

## 目的

- 学习C++新特性
- 学习STL源码
- 学习数据结构的实现原理
- 学习常用算法的实现原理

## 编译环境

- c++17
- mingw gcc 8.1.0
- cmake 3.10.2

## 参考书籍:

- STL源码剖析
- 算法导论
